The Clear Glass texture pack completely reworks and improves all glass blocks in Minecraft, making them truly transparent and aesthetic. No more annoying texture lines on the glass that obstruct the view ‚Äî only a clean and clear look.

![Example](https://github.com/user-attachments/assets/ad98cb00-08f4-429e-8062-e46c46fd6c44)

‚ú® Features

Stylistic Unity: The textures are made in a unified modern style that harmonizes with the vanilla aesthetic of the game.

Connected Textures Support (OptiFine): Use with OptiFine for a connected textures effect, creating the impression of a single, large window.

üì• Installation
Method 1: Standard Installation (without OptiFine)
Download the latest version of the .zip archive from the Releases section.

Open Minecraft, go to Options -> Resource Packs -> Open resource pack folder.

Copy the downloaded .zip file into the opened folder.

Return to the game, select "Clear Glass" in the pack list, and apply it.

Method 2: With OptiFine (recommended for the full effect)
Make sure you have OptiFine installed.

Repeat steps 1-4 from Method 1.

Shader Packs: Works with all popular shaders (BSL, SEUS, Complementary).
